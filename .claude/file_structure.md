# Project File Structure

| File/Folder | Description |
|-------------|-------------|
| `20180402-114759/` | Pre-trained FaceNet model files. |
| `attendance/` | Directory where attendance Excel sheets are generated. |
| `images/` | Static assets/images for documentation or UI. |
| `output/` | Dataset storage (subfolders named after individuals). |
| `user_interface.py` | Main GUI entry point (Tkinter). |
| `final_sotware.py` | Core algorithmic logic (Detection, Training, Recognition). |
| `facenet.py` | Utility functions for model loading and data processing. |
| `detect_face.py` | MTCNN implementation details. |
| `sheet.py` | Excel/Sheet management logic for attendance. |
| `face_aligner.py` | Logic for eye-based facial alignment. |
| `resizer.py` | Utility for image scaling. |
| `requirements.txt` | List of Python package dependencies. |
| `LICENSE` | Open-source licensing information. |
| `README.md` | Primary project documentation. |
| `classifier.pkl` | (Generated) Trained SVM model data. |
